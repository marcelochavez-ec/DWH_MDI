Descripcin y contexto

El proyecto Data Warehouse Analtico de la Direccin de Estadstica de la Seguridad tiene como objetivo la construccin de un Data Warehouse Analtico entre el Ministerio del Interior y Polica Nacional. Utilizando procesos de tipo ETL desarrollados en R, a travs del uso de los paquetes Tidyverse, RPostgreSQL, y programacin en R6, esto permitir la consolidacin de una Sistema de Datos para produccin estadstica.
